# Sprint 0 Ideation Folder

Thư mục này được sử dụng trong giai đoạn "Diverge" của Sprint 0 (Mô hình Double Diamond).

Mỗi thành viên trong team khi tham gia Hackathon sẽ tự động tạo một thư mục cá nhân tại đây bằng cách chạy lệnh `make setup-sprint0` ở Terminal. Thư mục sẽ tự động lấy tên theo tài khoản Github của bạn.

Bên trong thư mục đó, bạn có thể tự do tạo các file ghi chú, hoặc dùng AI (bằng lệnh chạy skill `ideation-assistant`) để brainstorm. AI sẽ đóng vai trò người phỏng vấn để giúp bạn đào sâu vào business logic, user journeys và tự động lưu file kết quả vào thư mục của bạn.

Sau khi tất cả các thành viên hoàn tất phần brainstorm cá nhân, Leader sẽ chạy skill `ideation-merge` để AI tự động quét tất cả các file trong các thư mục con ở đây và tổng hợp thành `SPRINT_0_NOTES.md`.
