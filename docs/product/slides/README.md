# Presentation Slides

## Format
HTML-based slides. The user will integrate their existing slide template into this directory.

## Placeholder
This directory is reserved for hackathon pitch decks and project presentations.

### Expected Files
- `index.html` — Main slide deck
- `assets/` — Images, diagrams, and media used in slides
- `styles.css` — Custom slide styling (if applicable)

## Usage
Open `index.html` in a browser to view the presentation.

## Tips for Hackathon Pitches
1. **Problem** — What pain point are you solving?
2. **Solution** — How does your AI-integrated app solve it?
3. **Demo** — Live demo or recorded walkthrough
4. **Architecture** — Show the system design (use diagrams from `docs/onboarding/01_ARCHITECTURE.md`)
5. **Tech Stack** — Highlight key technologies
6. **Impact** — Metrics, user feedback, or projected value
7. **Next Steps** — Roadmap and future features
